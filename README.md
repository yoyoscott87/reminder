# reminder

##  設定環境變數



setx REMINDER_SA_PATH "C:\Users\%USERNAME%\AppData\Roaming\ReminderApp\credentials.json"


將google服務帳戶用同一個 credentials處理，將google sheet新增工作表放提醒事項


## 執行jar檔

java -jar target\reminder-app-1.0-SNAPSHOT-shaded.jar
